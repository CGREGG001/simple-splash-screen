Bienvenue

Un projet simple pour expérimenter les splash screens, l’animation web, et un soupçon de design.

Fonctionnalités

  • Animation de logo fluide à l’ouverture
  • Effet de transition doux entre écran d’intro et contenu
  • Empêche le scroll pendant le splash screen
  • Structure claire du code (HTML / CSS / JS)

À propos du projet

Ce projet a été réalisé pour apprendre à créer des interfaces utilisateur animées avec des technologies simples. Il peut être intégré dans n’importe quel site web pour offrir une introduction dynamique et engageante.

Conçu avec ❤️ par un développeur junior passionné en technologie web
